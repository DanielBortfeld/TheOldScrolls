###################################

Start RougeLikeAttempt4.exe to run game

###################################

What you have to do:
Get through all levels without dying.
Enemies will follow you to the next level if you don't defeat them.
Gold and your remaining life points make up your score in the end.
gl&hf

###################################

Classic Controls:
Move: Arrow Keys (or on Numpad with 8456)
Open: O
Use : U
Zap : Z

One-handed Controls:
Move: WASD
Use/Open: E
Cast spell: C

Press Space to switch between Legend and Controls

###################################

Enemies:

Z = Zombie
G = Goblin
R = Giant Rat
